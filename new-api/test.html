Claude Code 使用教程

跟着这个教程，你可以轻松在自己的电脑上安装并使用 Claude Code，开启AI编程的全新体验

选择你的操作系统

根据你的系统选择对应的安装教程

1

安装 Node.js 环境

Claude Code 需要 Node.js 环境 才能运行，这是最重要的前置条件。

🖥️ Windows 安装方法

✅ 方法一：官网下载（推荐）
  1. 访问 https://nodejs.org/
  2. 点击 "LTS" 版本 进行下载
  3. 双击下载的 .msi 文件
  4. 按照安装向导完成安装，保持默认设置
⚡ 方法二：包管理器

适合有包管理器经验的用户：

# 使用 Chocolatey
choco install nodejs
# 或使用 Scoop
scoop install nodejs
💡 Windows 注意事项
  • 📌 推荐使用 PowerShell 而不是 CMD
  • 📌 权限问题： 遇到问题时以管理员身份运行
  • 📌 杀毒软件： 可能会误报，需要添加白名单
✅ 验证安装是否成功

打开 PowerShell，运行以下命令：

node --version
npm --version

如果显示版本号，说明安装成功了！

2

安装 Git Bash

Windows 环境下需要使用 Git Bash 安装 Claude Code。 安装完成后，环境变量设置和使用 Claude Code 仍然在普通的 PowerShell 或 CMD 中进行。

📦 下载并安装 Git for Windows

  1. 访问 https://git-scm.com/downloads/win
  2. 点击 "Download for Windows" 下载安装包
  3. 运行下载的 .exe 安装文件
  4. 保持默认设置，直接点击 "Next" 完成安装
🎯 安装完成后你会看到：
  • 📌 右键菜单： 在任意文件夹右键看到 "Git Bash Here" 选项
  • 📌 开始菜单： 可以从开始菜单启动 "Git Bash"
  • 📌 使用场景： 只需要在 Git Bash 中运行 npm install 命令
  • 📌 后续操作： 环境变量设置和使用都在 PowerShell/CMD 中
🔍 验证 Git Bash 安装

打开 Git Bash，输入以下命令：

git --version

如果显示版本号，说明安装成功！

3

安装 Claude Code

⬇️ 全局安装 Claude Code

⚠️ 重要提醒

必须在 Git Bash 中运行，不要使用 PowerShell！

打开 Git Bash，运行以下命令：

# 在 Git Bash 中全局安装 Claude Code
npm install -g @anthropic-ai/claude-code

这个命令会从 npm 官方仓库下载并安装最新版本的 Claude Code。

🔍 验证 Claude Code 安装

安装完成后，输入以下命令检查：

claude --version

如果显示版本号，恭喜你！Claude Code 已经成功安装了。

4

设置环境变量

为了让 Claude Code 连接到你的中转服务，需要设置 两个环境变量。

⚙️ 配置 Claude Code 环境变量

🚀 方法一：PowerShell 临时设置（推荐）

在 PowerShell 中运行以下命令：

$env:ANTHROPIC_BASE_URL = "https://new.yunai.link/"
$env:ANTHROPIC_AUTH_TOKEN = "你的API密钥"

💡 记得将 "你的API密钥" 替换为在 API Keys 页面中创建的实际密钥。

💾 方法二：系统环境变量（永久设置）
  1. 右键"此电脑" → "属性" → "高级系统设置"
  2. 点击"环境变量"按钮
  3. 在"用户变量"中点击"新建"
  4. 添加以下两个变量：
变量名： ANTHROPIC_BASE_URL
变量值： https://new.yunai.link/
变量名： ANTHROPIC_AUTH_TOKEN
变量值： 你的API密钥
🔍 验证环境变量设置

设置完环境变量后，通过以下命令验证是否设置成功：

在 PowerShell 中验证：
echo $env:ANTHROPIC_BASE_URL
echo $env:ANTHROPIC_AUTH_TOKEN
在 CMD 中验证：
echo %ANTHROPIC_BASE_URL%
echo %ANTHROPIC_AUTH_TOKEN%

📋 预期输出示例：

https://new.yunai.link/
cr_xxxxxxxxxxxxxxxxxx

💡 如果输出为空或显示变量名本身，说明环境变量设置失败，请重新设置。

5

开始使用 Claude Code

🚀 启动和使用

现在你可以开始使用 Claude Code 了！

🎯 启动 Claude Code
claude
📁 在特定项目中使用
# 进入你的项目目录
cd C:\path\to\your\project
# 启动 Claude Code
claude

🔧 Windows 常见问题解决

⚠️ 安装时提示 "permission denied" 错误

这通常是权限问题，尝试以下解决方法：

  • 以管理员身份运行 PowerShell
  • 或者配置 npm 使用用户目录：npm config set prefix %APPDATA%\npm
🛡️ PowerShell 执行策略错误

如果遇到执行策略限制，运行：

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
⚙️ 环境变量设置后不生效

设置永久环境变量后需要：

  • 重新启动 PowerShell 或 CMD
  • 或者注销并重新登录 Windows
  • 验证设置：echo $env:ANTHROPIC_BASE_URL

  • 📌 终端选择： 建议使用 Terminal 或 iTerm2

设置环境变量

为了让 Claude Code 连接到你的中转服务，需要设置 两个环境变量。

⚙️ 配置 Claude Code 环境变量

🚀 方法一：临时设置（当前会话）

在 Terminal 中运行以下命令：

export ANTHROPIC_BASE_URL="https://new.yunai.link/"
export ANTHROPIC_AUTH_TOKEN="你的API密钥"

💡 记得将 "你的API密钥" 替换为实际密钥。

💾 方法二：永久设置

编辑你的 shell 配置文件：

# 对于 zsh (默认)
echo 'export ANTHROPIC_BASE_URL="https://new.yunai.link/"' >> ~/.zshrc
echo 'export ANTHROPIC_AUTH_TOKEN="你的API密钥"' >> ~/.zshrc
source ~/.zshrc
# 对于 bash
echo 'export ANTHROPIC_BASE_URL="https://new.yunai.link/"' >> ~/.bash_profile
echo 'export ANTHROPIC_AUTH_TOKEN="你的API密钥"' >> ~/.bash_profile
source ~/.bash_profile

🔧 macOS 常见问题解决

尝试以下解决方法：

  • 使用 sudo 安装：sudo npm install -g @anthropic-ai/claude-code
  • 或者配置 npm 使用用户目录：npm config set prefix ~/.npm-global
🛡️ macOS 安全设置阻止运行

如果系统阻止运行 Claude Code：

  • 打开"系统偏好设置" → "安全性与隐私"
  • 点击"仍要打开"或"允许"
  • 或者在 Terminal 中运行：sudo spctl --master-disable
⚙️ 环境变量不生效

检查以下几点：

  • 确认修改了正确的配置文件（.zshrc 或 .bash_profile）
  • 重新启动 Terminal
  • 验证设置：echo $ANTHROPIC_BASE_URL

🐧 Linux 安装方法

✅ 方法一：使用官方仓库（推荐）

# Ubuntu/Debian

# CentOS/RHEL/Fedora
sudo dnf install nodejs npm

  • 📌 权限问题： 如果遇到权限问题，使用 `sudo`
